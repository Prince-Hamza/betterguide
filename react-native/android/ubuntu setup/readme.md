
#### To setup react native correctly you need 4 things
1. OpenJdk
2. Android Sdk
3. Android Sdk build tools
4. Platform-tools

#### update references
```
sudo apt update
```

#### install open jdk 11 (used by latest Android Studio (Bumblebee)
```
sudo apt install openjdk-11-jdk

```

#### check version
```
java -v
```
